# Excel-tool
# 📊 Excel 数据处理工具

一个带图形界面的 Excel/CSV 文件处理工具，支持多文件合并、数据清洗、分组汇总，并提供拖拽操作和进度提示，让数据处理变得简单高效

---

## ✨ 功能特点

- 📂 **多文件合并** – 支持选择多个 Excel（.xlsx, .xls）或 CSV 文件，一键合并成一张总表，并自动添加“来源文件”列。
- 🧹 **数据清洗** – 删除完全空白行、删除重复行、自定义删除列，让数据更规范。
- 📊 **分组汇总** – 按任意列分组统计，可计算数据量、总和、平均值，输出汇总报表。
- 🖱️ **拖拽操作** – 支持直接将文件或文件夹拖入窗口，自动递归查找所有支持的 Excel/CSV 文件。
- 📝 **实时日志** – 右侧日志区域实时显示处理进度和详细操作记录。
- ⏳ **进度提示** – 处理大数据时显示进度条，界面不卡顿。
- 💾 **记住输出目录** – 自动保存上次使用的文件夹路径，方便重复操作。
- 🚀 **打包即用** – 可打包成独立 EXE 文件，无需安装 Python 环境。
